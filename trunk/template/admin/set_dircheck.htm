使用帮助
本功能用于检验程序正常运行所必须的目录是否具备相应的权限，如果您是Unix类主机，请通过FTP软件把相应目录设置为777属性，如果您的空间为Windows，则请检查文件夹的权限设置。
检验信息
目录名称 路径 是否可写
静态文件发布目录 $sys[htmdir] $htm_check
附件资源上传目录 $sys[attachdir] $img_check
图片缩略图保存目录 {$sys[attachdir]}/s $imgs_check
图片本地化临时目录 {$sys[attachdir]}/temp $imgtemp_check
用户自定义模板目录 {$user_tplpath} $usertpl_check
广告JS缓存目录 script/cms $jscache_check
数据资料保存目录 data $data_check
模板编译缓存目录 data/tpl_cache $tpl_check
动态查询数据缓存 data/sql $sql_check
RSS文件缓存目录 data/rss $rss_check
缓存资料保存目录 data/cache $cache_check
